# Linux Shell Scripting Lab

## Objective
Practiced basic Bash scripting concepts including shebang usage, executable permissions, variables, loops, conditional logic, and searching log files for a specific keyword.

## Skills Demonstrated
- Bash scripting
- Linux command line navigation
- File permissions using chmod
- Script execution
- for loops
- if/elif/else conditionals
- grep log searching
- Working with /var/log files
